## 💡 兼职策略

### 3个原则

1. **做减法，不做加法** - 功能宁少勿多
2. **快速验证，不求完美** - 先跑起来再优化
3. **控制成本，留有余地** - 失败了不能伤筋动骨

### 技术选型

**优先选择**:
- 快速开发框架 (FastAPI + React/Flutter)
- 现成UI模板
- 云服务 (减少运维)
- GPT-4 API (而非自己训练)

**避免**:
- 复杂架构
- 自建服务器
- 过度优化
