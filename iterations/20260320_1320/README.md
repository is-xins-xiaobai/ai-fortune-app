### 1.2 三大内容类型

#### 类型A: 知识内容 (建立专业度)

**目标**: 建立命理/心理专业形象
**平台**: 知乎/公众号/小红书
**占比**: 30%

**子类型**:

**A1: 命理科普**
- 八字基础入门
- 五行相生相克
- 十二生肖运势
- 紫微斗数简介

**示例标题**:
- "八字里的'食神'是什么意思？"
- "2025年犯太岁的生肖要注意什么？"
- "五行缺什么怎么补？"

**A2: 心理成长**
- 情绪管理技巧
- 职场心理建设
- 人际关系处理
- 自我认知提升

**示例标题**:
- "如何缓解职场焦虑？"
- "INFJ适合什么工作？"
- "怎样建立自信？"

**A3: AI+命理结合**
- AI算命原理
- 传统vs现代命理
- 科技赋能传统文化

**示例标题**:
- "AI真的能算命吗？"
- "传统命理师的末日？"
- "AI算命 vs 人工算命"

---

#### 类型B: 情感内容 (建立共鸣)

**目标**: 引发情感共鸣，建立用户连接
**平台**: 小红书/抖音/朋友圈
**占比**: 50%

**子类型**:

**B1: 用户故事**
- 真实用户使用体验
- 深夜倾诉场景
- 决策辅助案例

**示例标题**:
- "凌晨2点，我和AI聊了半小时"
- "纠结要不要辞职，AI帮我理清了思路"
- "比闺蜜还懂我的AI"

**B2: 情感共鸣**
- 职场焦虑
- 情感困惑
- 人生迷茫

**示例标题**:
- "25岁，工作3年，还是不知道自己想要什么"
- "有没有人和我一样，明明很累却睡不着？"
- "成年人的崩溃，都是无声的"

**B3: 治愈内容**
- 温暖文案
- 励志语录
- 生活小确幸

**示例标题**:
- "今天也要对自己说：你已经很棒了"
- "不必着急，每个人都有自己的时区"
- "慢慢来，比较快"

---

**目标**: 提升互动率，增加曝光
**平台**: 小红书/微博/社群
**占比**: 20%

**子类型**:

**C1: 话题讨论**
- MBTI话题
- 星座话题
- 职场话题

**示例标题**:

- 运势测试

**示例标题**:

**C3: 活动/福利**

**示例标题**:

### 2.2 内容模板库

建立模板库，提高生产效率：

**模板A: 用户故事型**
```
标题: [时间] + [场景] + [感受]

开头: 场景描述 (深夜/焦虑时刻)
中间: 具体故事 (和AI的对话)
高潮: 转折/感悟
结尾: 产品软性植入

#AI算命 #深夜emo #职场焦虑
```

**模板B: 干货知识型**
```
标题: [问题] + [答案暗示]

开头: 痛点引入
要点1: 知识点1
要点2: 知识点2
要点3: 知识点3
结尾: 引导关注/试用

#命理知识 #八字 #自我成长
```

**模板C: 情感共鸣型**
```
标题: [情绪词] + [群体]

文案: 简短有力的情感文案
配图: 温暖/治愈系图片
互动: 提问引导评论

#治愈 #情感共鸣 #深夜树洞
```

## 📊 内容效果评估

### 4.1 核心指标

#### 曝光指标
| 指标 | 定义 | 健康值 |
|------|------|--------|
| 阅读量/播放量 | 内容被浏览次数 | >1000 |
| 曝光量 | 被推荐次数 | >5000 |
| 完播率 | 看完内容的比例 | >30% |

#### 互动指标
| 指标 | 定义 | 健康值 |
|------|------|--------|
| 点赞率 | 点赞/阅读 | >3% |
| 评论率 | 评论/阅读 | >1% |
| 收藏率 | 收藏/阅读 | >2% |
| 分享率 | 分享/阅读 | >0.5% |

#### 转化指标
| 指标 | 定义 | 健康值 |
|------|------|--------|
| 点击率 | 点击链接/阅读 | >5% |
| 关注率 | 关注/阅读 | >1% |
| 转化率 | 注册/点击 | >20% |

### 4.2 内容效果分析

**爆款内容特征**:
- 标题有冲突/好奇/共鸣
- 开头3秒抓住注意力
- 内容有情绪价值
- 结尾有互动引导

**优化方向**:
| 问题 | 优化策略 |
|------|---------|
| 曝光低 | 优化标题/封面，蹭热点 |
| 点击低 | 优化标题，增加好奇心 |
| 互动低 | 增加互动引导，提问 |
| 转化低 | 优化CTA，增加紧迫性 |

### 4.3 A/B测试框架

**测试元素**:
- 标题 (疑问句 vs 陈述句)
- 封面 (人像 vs 文字)
- 发布时间 (早上 vs 晚上)
- CTA (直接 vs 软性)

**测试方法**:
- 同一内容，不同版本
- 相似时间段发布
- 样本量 >1000曝光
- 观察3天数据

## 💡 内容创作技巧

### 5.1 标题写作公式

**公式1: 数字+结果**
- "用了30天，我的焦虑减轻了80%"
- "3个方法，帮你理清职业规划"

**公式2: 疑问+悬念**
- "AI真的能算命吗？我试了一下..."
- "为什么你总是感到焦虑？"

**公式3: 情绪+共鸣**
- "成年人的崩溃，都是无声的"
- "25岁，我终于不再迷茫了"

**公式4: 对比+冲突**
- "AI算命 vs 人工算命，哪个更准？"
- "月薪3000和月薪30000的区别"

### 5.2 开头钩子技巧

- **场景化**: "凌晨2点，我躺在床上..."
- **提问式**: "你有没有过这样的时刻..."
- **冲突式**: "所有人都说我应该..."
- **数据式**: "据统计，80%的25岁..."

### 5.3 结尾CTA设计

- **软性引导**: "如果你也有类似的困惑..."
- **互动提问**: "你怎么看？评论区聊聊"
- **福利诱导**: "私信'体验'，免费试用"
- **关注引导**: "关注我，了解更多"
