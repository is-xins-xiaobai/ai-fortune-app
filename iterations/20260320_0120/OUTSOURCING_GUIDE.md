## 🎯 外包策略

### 为什么要外包？

1. **节省时间** - 2人兼职团队，时间有限
2. **专注核心** - 后端和AI逻辑是核心竞争力
3. **质量保证** - 专业设计师和前端开发
4. **成本可控** - ¥11000-17000，在预算内

### 外包范围

**外包**:
- ✅ UI设计 (3-5天，¥3000-5000)
- ✅ 前端开发 (7-10天，¥8000-12000)

**不外包** (自己开发):
- ❌ 后端API
- ❌ AI对话逻辑
- ❌ 八字计算引擎
- ❌ 知识库构建

## 👨‍🎨 外包设计师

### 需求描述

**项目名称**: AI命理咨询App - UI设计  
**预算**: ¥3000-5000  
**时间**: 3-5天

**交付物**:
1. 首页设计 (用户注册/登录)
2. 命盘展示页 (八字排盘可视化)
3. 对话界面 (聊天界面)
4. 个人中心页
5. 支付页

**设计要求**:
- 风格: 现代简约 + 东方元素
- 色调: 暖色系 (奶茶色、暖橙色)
- 感觉: 温暖、专业、陪伴感
- 参考: Replika (AI陪伴App) + 测测星座

**技术规格**:
- 设计工具: Figma (优先) 或 Sketch
- 交付格式: Figma源文件 + 切图
- 响应式: 适配手机端 (375px宽度)

---

**优先级1: 猪八戒网/一品威客**
- 优点: 平台担保，有保障
- 缺点: 价格较高
- 预算: ¥4000-5000

**优先级2: 朋友圈/微信群**
- 优点: 信任度高，沟通方便
- 缺点: 需要人脉
- 预算: ¥3000-4000

**优先级3: 站酷/UI中国**
- 优点: 设计师质量高
- 缺点: 需要筛选
- 预算: ¥4000-6000

## 👨‍💻 外包前端开发

### 需求描述

**项目名称**: AI命理咨询App - 前端开发  
**预算**: ¥8000-12000  
**时间**: 7-10天

**交付物**:
1. 前端代码 (React)
2. 与后端API联调
3. 响应式适配
4. 基础测试

**技术栈**:
- 框架: React 18
- UI库: Ant Design Mobile 或 TDesign
- 状态管理: Redux Toolkit
- HTTP客户端: Axios
- 构建工具: Vite

**页面清单**:
1. 注册/登录页
2. 出生信息录入页
3. 命盘展示页
4. 对话页面 (核心)
5. 个人中心页
6. 支付页面

---

- 预算: ¥10000-12000
