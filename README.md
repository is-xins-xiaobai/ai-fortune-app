## 💎 核心价值主张

**我们不是"AI算命App"，我们是：**

### 1. AI心理成长伙伴
- 用命理语言做心理咨询
- 24/7无评判的倾听者
- 长期陪伴，而非一次性工具

### 2. 真正的差异化
| 维度 | 竞品 | 我们 |
|------|------|------|
| 定位 | 工具/娱乐 | **心理伙伴** |
| 价值 | 预测/排盘 | **陪伴+成长** |
| 关系 | 一次性 | **长期陪伴** |
| 频次 | 偶尔 | **每天互动** |

### 3. 可持续的商业模式
- 订阅制为主 (¥29/月)
- 第一年目标: ¥480万收入
- 付费转化率: 8%
- 用户粘性高，LTV大
